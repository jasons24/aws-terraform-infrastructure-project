# AWS Infrastructure Automation with Terraform

This project demonstrates Infrastructure as Code (IaC) using Terraform to provision and manage AWS resources.

## Technologies Used

- AWS
- Terraform
- EC2
- VPC
- IAM
- CloudWatch
- Linux
- Git & GitHub

## Project Goals

- Automate AWS infrastructure deployment
- Configure Linux-based web server hosting
- Implement monitoring using CloudWatch
- Follow Infrastructure as Code best practices
